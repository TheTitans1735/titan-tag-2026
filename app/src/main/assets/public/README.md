# מדריך התקנה למערכת ארכיאולוגיה שטח

## סקירת המערכת
מערכת מבוססת אינטרנט לניהול ממצאים ארכיאולוגיים בשטח, מותאמת לשימוש במובייל עם ממשק עברי מלא ותמיכה ב-RTL.

## צעדי התקנה

### 1. הגדרת Google Sheets
1. צור גיליון חדש ב-Google Sheets
2. קרא לו "מערכת ארכיאולוגיה"
3. צור 4 גיליונות עם השמות הבאים:
   - משתמשים
   - אתרים
   - ממצאים
   - תמונות

### 2. הגדרת עמודות בכל גיליון
בצע את הגדרת העמודות לפי קובץ `google-sheets-setup.md`

### 3. הגדרת Google Apps Script
1. בגיליון ה-Google Sheets, לחץ על "תוספים" → "Apps Script"
2. מחק את הקוד הקיים
3. הדבק את הקוד מקובץ `google-apps-script.js`
4. לחץ על "שמור" (סמל דיסקית)
5. בתפריט הנגישות, בחר "פרסום" → "פריסה כאפליקציית אינטרנט"
6. בחר "ביצוע כמו: אני" ו"מי יכול לגשת: כולם, אפילו אנונימיים"
7. לחץ "פרסם"
8. העתק את כתובת ה-URL שתתקבל

### 4. הגדרת האפליקציה
1. פתח את קובץ `index.html`
2. בסוף הקובץ, מצא את השורה:
   ```javascript
   const scriptUrl = 'YOUR_GOOGLE_APPS_SCRIPT_URL';
   ```
3. החלף את הכתובת בזו שקיבלת מ-Google Apps Script

### 5. העלאת הקבצים לשרת
העלה את כל הקבצים לשרת אינטרנט או לשירות אירוח כמו:
- GitHub Pages
- Netlify
- Firebase Hosting
- או שרת אינטרנט פרטי

## קבצים נדרשים
- `index.html` - האפליקציה הראשית
- `google-sheets-setup.md` - הגדרת גיליונות
- `google-apps-script.js` - קוד החיבור ל-Google Sheets

## תכונות מערכת
- ✅ מסך כניסה עם פרטי משתמש חד-פעמיים
- ✅ הפרדת הרשאות (מנהל/עובד)
- ✅ ממשק עברי מלא עם תמיכת RTL
- ✅ סריקת ברקוד/QR לזיהוי ממצאים
- ✅ העלאת תמונות מהמצלמה
- ✅ רקע ורוד
- ✅ מיקום GPS אוטומטי
- ✅ שמירת נתונים ב-Google Sheets

## הוראות שימוש

### למשתמשים ראשונים:
1. היכנסו לכתובת האפליקציה
2. מלאו פרטים אישיים במסך הכניסה
3. הפרטים יישמרו מקומית במכשיר

### לעובדים:
- יכולים להוסיף ממצאים רק באתר שהוקצה להם
- יכולים לסרוק קודי QR של ממצאים קיימים
- יכולים לצלם תמונות ולצרף לממצאים

### למנהלים:
- יכולים לנהל אתרים וחלקות
- יכולים לגשת לכל המידע במערכת
- יכולים להגדיר משתמשים חדשים

## טיפים חשובים
- ודאו שה-Google Sheets משותף עם "כל מי שיש לו את הקישור"
- השתמשו בכתובת ה-HTTPS כדי לאפשר גישה למצלמה ו-GPS
- בדקו את ההרשאות בדפדפן לגישה למצלמה ומיקום
- שמרו גיבוי של קובץ ה-Google Sheets

## תמיכה
במקרה של בעיות:
1. בדוק את כתובת ה-URL של Google Apps Script
2. ודא שכל העמודות בגיליונות נקראות בשמות הנכונים
3. בדוק את הרשאות הגישה בדפדפן
4. ודא שהמכשיר תומך ב-HTML5 ו-GPS
